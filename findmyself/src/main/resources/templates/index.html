'나혼자 찾는다'

나에게 알맞은 동네 찾기

서울이 낯선 사용자를 대상으로 조건에 부합한 동네를 추천 및 분석해드립니다.
예산, 교통, 편의, 거주 환경 등의 조건을 설정하면 적합한 동네를 추천 및 분석해드립니다!
지금 바로 이용해보세요!

기능 소개

사용자의 조건에 맞는 거주지 추천

조건의 매칭률에 따라 여러 거주지를 추천 해드립니다.

매칭정도가 높을 수록 진한색으로 알려드릴게요!

카테고리별 그룹화 정보

각 특성이 유사한 동네별로 군집화된 결과를 보여드립니다.

상세한 분석결과

카테고리별 상세 분석 정보를 보기 쉽게 제공합니다.

교통 정보

직장/학교까지의 자동차, 대중교통 정보를 제공합니다.

해당 경로의 분석 결과와 출퇴근 시간대의 혼잡도를 알 수 있습니다.

생활 만족도, 매칭 정보

실 거주자의 생활 만족도와 조건의 매칭정도를 제공합니다.

추가 동네 추천

설정한 학교/직장과의 거리를 고려한 동네와

현재 추천된 곳과 비슷한 특징을 가진 동네를 추천해줍니다.

사용된 API 소개

카카오맵 API

카카오맵 api를 활용하여 서울시 지도 위에 마커, 경로 표시, 특정 지역 강조, 도로명 주소의 좌표 변환등을 하였습니다.

카카오 로그인 API

카카오 계정과 연동하여 간단한 로그인과 회원가입이 가능합니다.

Tmap API

Tmap API를 사용하여 목적지와 추천 동네 사이의 자동차 경로 검색이 가능합니다.

오디세이 API

오디세이 API를 사용하여 목적지와 추천 동네 사이의 대중 교통 경로 검색이 가능합니다.
